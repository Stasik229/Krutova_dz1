Форматирование

«address»

Тегом «address» размечается контактная информация о человеке, людях или организации.

«blockquote»

Для вставки большой цитаты на страницу есть специальный семантический тег «blockquote».
Слово большой выделено, потому что для цитаты, которая скрывается в дебрях текста (внутристрочная) есть свой отдельный тег — «q». Но сейчас не об этом.

«q»

Тег «q» пригодится в ситуации, когда нужно вставить цитату другого человека прямо в текст предложения.

«wbr»

Тег «wbr» указывает браузеру место, где допускается делать перенос строки в тексте, если текст не влезает в родительский элемент.
Это полезно для адаптивной вёрстки если у вас в тексте есть длинные слова или, например, ссылки.

«hr»

Та самая горизонтальная черта, которая разделяет смысловые блоки на странице.

«b»

Тег «b» используется для привлечения внимания к части текста без добавления какой-либо важности или изменения тона: например, если мы хотим визуально выделить ключевые слова, названия продуктов в обзорах, вводные части статей.

«i»

Тег «i» используется для изменения голоса или настроения части текста, чтобы передать его другое качество: официальное название, идиома из другого языка, транслитерация или ремарки.
Для выделения важного используется «strong», для изменения тона — «em».
По умолчанию браузеры применяют к «i» курсивное начертание с помощью font-style: italic.

«s»

Тег «s» обозначает содержимое, которое уже не корректно или потеряло актуальность.
